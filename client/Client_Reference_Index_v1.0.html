How to Use This Document Pack

All reference HTML pages are stored in the same folder as this index file. Click any document title below to open the linked page in your browser.

Keep the full set together when sharing with stakeholders — links use relative filenames and will not resolve if files are moved apart.

Recommended Reading Order

For first-time client review of the Phase 0 blueprint package:

  1. Project Master Overview — Project hub — phases, screen inventory, prototype links, and client dependencies.
  2. Phase 0 Blueprint Acceptance Form — Blueprint attestation, scope checklist, and Public Inquiry decision — for client signing.
  3. Lead Lifecycle Summary — Six-stage lead pipeline, allowed transitions, entry paths, and agent actions.
  4. Software Requirements Specification — Authoritative requirements baseline for Phases 0–3 (functional, data, integrations, acceptance).
  5. Phase 0 Blueprint Deliverable — Phase 0 deliverable status, Figma links, and full screen inventory.
  6. Deployment Checklist — Pre-deployment checklist, go-live gates, security hardening, and handover items.
