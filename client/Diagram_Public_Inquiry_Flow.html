Public Website Inquiry Flow

Alsultania Travel CRM & Management System

flowchart TD
  A[Customer opens public website /inquiry] --> B[Enter mobile number]
  B --> C[System sends 6-digit SMS OTP]
  C --> D[Customer enters OTP]
  D --> E{OTP valid?}
  E -->|No| F[Show error / allow resend after cooldown]
  F --> D
  E -->|Yes| G[Session marked phone-verified]
  G --> H[Public inquiry form loads]
  H --> I[Customer fills required fields: name, service type, type fields, dates/travelers]
  I --> J[Optional: view Previous Inquiries for verified phone]
  J --> K[Click Submit Inquiry]
  K --> L[System validates required fields + saves lead]
  L --> M["lead_via = Website · assign next available agent · created_by = null"]
  M --> N[Confirmation screen with reference ID]
  M --> O[Assigned agent notified in CRM inbox]
